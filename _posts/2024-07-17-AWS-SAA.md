## 1. AWS SAA란?

**SAA**는 *AWS Certified Solutions Architect – Associate*의 약자로, AWS 서비스를 사용해 **확장 가능·고가용·비용 효율적인 아키텍처를 설계·배포하는 능력**을 검증하는 자격증이다.

![Untitled](/assets/img/Infra/AWS/SAA/Untitled%201.png)

| 검증 목적 | 내용 |
|------|------|
| **아키텍처 설계** | AWS 인프라를 설계·배포하는 기술 역량 |
| **모범 사례** | 안정적·안전한 클라우드 솔루션 설계 |
| **서비스 통합** | 여러 AWS 서비스를 결합해 솔루션 구현 |

### 시험 구성

| 항목 | 내용 |
|------|------|
| 질문 유형 | 다지선다형 · 다중 응답형 |
| 시험 시간 | **130분** |
| 합격 점수 | 1000점 만점 중 **720점 이상** |
| 언어 | 영어 · 일본어 · 한국어 · 중국어 등 |

## 2. 공부 방법

### ☑️ 개념 및 실습

여러 교육기관의 **정부지원 재직자 교육 프로그램** 중 한 곳을 골라, 평일엔 회사에서 일하고 주말엔 그곳에서 **AWS 실습**을 했다. 그래서 흔히 보는 유데미 강의나 책은 따로 보지 않았다.

### ☑️ 덤프.. 그리고 덤프..

넉넉하게 **총 2달** 정도 계획을 잡고, 하루 2시간씩 **덤프 문제**를 계속 풀었다. 덤프는 검색으로 구할 수도 있지만 마음 편하게 구매했다. 총 **936문제**였고, **1회독 + 오답 0.2회독** 정도 하고 시험을 봤다.

> 💡 SAA는 실습으로 개념을 잡되, 마무리는 **덤프 반복**이 핵심이다. 실제로 시험 문제 대부분이 덤프에서 봤던 유형과 거의 일치했다.

---

## 3. 후기 및 팁

### ☑️ 소감

AWS 자격증 시험은 **지독한 덤프와의 싸움**이라 후반에 정말 지친다. 문제만 봐도 한숨이 나오고, 계획했던 전체 2회독은 하지도 못한 채 시험을 봐서 많이 힘들 거라 생각했다. 그런데 그날 저녁 **합격 메일**을 받고 나니 그동안의 노고가 싹 사라졌다. 힘들더라도 **꾸준히 덤프를 도는 것**이 정답이다.

### 📌 팁

> - 영어가 모국어가 아닌 사람은 **시험 편의 요청**으로 약 **30분**을 추가로 받을 수 있다.
> - 덤프 많이 보세요, 덤프!
