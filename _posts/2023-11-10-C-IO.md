## 📌 들어가며

지금까지는 값을 코드 안에 직접 써넣었다. 이제는 **사용자가 직접 값을 입력**하도록 만들어 보자. 이번 글에서는 입력 함수 **`scanf`**와 문자열 입력 **`gets`**를 정리한다.

> **입출력(I/O)이란?**
>
> | 구분 | 기본 장치 | 그 외 |
> |------|------|------|
> | **입력(input)** | 키보드(standard input) | 마우스, 네트워크, 파일… |
> | **출력(output)** | 모니터(standard output) | 프린터, 네트워크, 파일… |

C에서는 **`scanf`** 문장으로 사용자가 원하는 값을 변수에 입력할 수 있다.

---

## 1. scanf로 값 입력받기

국어·영어·수학 점수를 입력받아 보자.

```c
#pragma warning(disable:4996)   // scanf 에러 무시 (Visual Studio)
...
int kor, eng, math;
scanf("%d %d %d", &kor, &eng, &math);   // 정수 3개, 공백으로 구분
printf("국어=%d 영어=%d 수학=%d\n", kor, eng, math);
```

![Desktop View](/assets/img/Programming-Language/C/IO/1.png){: width="100%" }

여기서 두 가지가 핵심이다.

| 포인트 | 설명 |
|------|------|
| **`&` (주소 연산자)** | `printf`와 달리, `scanf`는 입력할 변수명 앞에 `&`를 붙인다 |
| **공백으로 구분** | `%d`를 여러 개 두면 입력 시 공백으로 나눠 넣는다 |
| **`#pragma warning(disable:4996)`** | Visual Studio에서 `scanf` 관련 에러를 무시 (자세한 내용은 추후) |

![Desktop View](/assets/img/Programming-Language/C/IO/2.png){: width="100%" }

각각 `50 70 90`을 입력하고 엔터를 치면?

![Desktop View](/assets/img/Programming-Language/C/IO/3.png){: width="100%" }

다음 문장인 `printf`가 입력받은 값으로 출력된다.

> 💡 `scanf`에서 변수명 앞에 `&`를 붙이는 이유는 "값을 저장할 **주소**"를 알려주기 위함이다. 지금은 규칙으로 외우고, 포인터를 배우면 자연스럽게 이해된다.

## 2. 문자열 입력 — `gets`

그렇다면 문자열(이름 등)은 어떻게 입력받을까?

![Desktop View](/assets/img/Programming-Language/C/IO/4.png){: width="100%" }

`scanf("%s", ...)`로 문자열을 받으면 **띄어쓰기를 포함할 수 없다**(공백에서 입력이 끊긴다). 띄어쓰기까지 포함한 풀네임을 받으려면 **`gets(변수명)`**을 쓴다.

```c
char name[20];
gets(name);          // 띄어쓰기 포함 입력 가능
printf("%s\n", name);
```

![Desktop View](/assets/img/Programming-Language/C/IO/6.png){: width="100%" }

`Shin Dong Hee`처럼 공백이 포함된 문자열도 잘 출력된다.

## 📝 정리

| 함수 | 용도 | 특징 |
|------|------|------|
| **`scanf`** | 값 입력 | 변수 앞에 `&` 필요, 공백으로 여러 값 구분 |
| **`gets`** | 문자열 입력 | 띄어쓰기 포함 한 줄 입력 |

입력의 핵심은 **`scanf`에서는 변수 앞에 `&`를 붙인다**는 점이다. 띄어쓰기가 포함된 문자열은 `gets`를 쓴다는 것도 함께 기억하자.
