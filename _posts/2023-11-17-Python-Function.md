## 📌 들어가며

이번 글에서는 파이썬의 **함수(function)**를 정리한다. 함수는 코드를 정리하고 **재사용성**을 높이는 도구다.

> **`def` 키워드**는 "함수를 정의한다"는 것을 파이썬 인터프리터에게 알리는 역할을 한다.

![Desktop View](/assets/img/Programming-Language/Python/Function/1.png)

---

## 1. 기본 함수

```python
def 인사하기():
    print("안녕하세요")
    print("반갑습니다")
    print("내이름은 무지!")

인사하기()
# >>> 안녕하세요
# >>> 반갑습니다
# >>> 내이름은 무지!
```

출력문을 일일이 반복할 필요 없이 **함수 호출 한 번**으로 재사용한다.

---

## 2. 파라미터를 갖는 함수

```python
def 청첩장(신랑, 신부):
    print("-" * 20)
    print(" 청 첩 장 ")
    print("-" * 20)
    print(f"{신랑} ♡ {신부} 결혼을 축하합니다")

청첩장("희철", "영희")
# >>> 희철 ♡ 영희 결혼을 축하합니다
```

리스트 형태로 받을 수도 있다.

```python
def 청첩장(고객):
    print(f"{고객[0]} ♡ {고객[1]} 결혼을 축하합니다")

청첩장(["희철", "영희"])
```

> 💡 파라미터를 어떻게 설계하느냐는 **개발자의 선택**이다. 다만 정의한 방식대로 호출 시 값을 넣어야 오류가 안 난다.

## 3. 반환값이 있는 함수

지금까지는 함수 내부에서 값을 출력했지만, `len()`처럼 **결과를 반환**하면 그 값을 출력할지 연산에 쓸지 **사용자가 결정**할 수 있다.

```python
def 함수():
    변수 = 100
    return 변수      # return을 만나면 값 반환 후 함수 종료

결과 = 함수()
print(결과)
# >>> 100
```

> 💡 `return`을 만나면 **그 값을 반환하고 함수는 즉시 종료**된다(아래 코드가 있어도 실행 안 됨). `함수()` 전체가 100이라는 값이 되어, 이를 `결과`에 담아 활용할 수 있다.

## 📝 정리

```
파이썬 함수
├─ 정의    def 이름(): ...
├─ 파라미터  def 이름(a, b): 호출 시 값 전달
└─ 반환      return 값 → 함수를 값처럼 활용, 만나면 종료
```

| 개념 | 한 줄 정의 |
|------|------|
| **def** | 함수를 정의하는 키워드 |
| **파라미터** | 함수가 받는 입력값 |
| **return** | 결과 반환 + 함수 종료 |

함수는 "코드를 묶어 재사용"하는 기본 도구다. 특히 `return`이 있는 함수는 결과를 출력하든 연산에 쓰든 자유롭게 활용할 수 있다. 다음엔 클래스를 다룬다.
