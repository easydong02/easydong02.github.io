## 📌 들어가며

이번 글에서는 파이썬의 **반복문(for)**을 정리한다. 파이썬의 for는 C/Java와 달리 **자료구조의 데이터를 하나씩 꺼내 순회**하는 방식이라 직관적이다.

> **반복문이란?**
> 자료구조의 데이터 수만큼 코드를 반복 실행. "데이터를 하나씩 **바인딩**하고 들여쓰기된 코드를 실행"하는 과정을 반복한다.

---

## 1. 기본형

```python
for 변수 in 자료구조:   # 끝에 콜론(:) 주의
    실행할 코드         # 들여쓰기
```

```python
리스트 = ['철수', '영희', '희철']

for name in 리스트:
    print(name)
# >>> 철수
# >>> 영희
# >>> 희철
```

> 💡 `for 변수 in 자료구조:`에서 **콜론(`:`)**과 **들여쓰기**가 필수다. 파이썬은 중괄호 대신 들여쓰기로 코드 블록을 구분한다.

### 슬라이싱으로 범위 제한

```python
for name in 리스트[:2]:   # 인덱스 0~1만
    print(name)
# >>> 철수
# >>> 영희
```

## 2. 조건과 함께 쓰기

`if`를 결합하면 원하는 값만 걸러낼 수 있다.

```python
for name in 리스트:
    if "희철" in name:
        print(name)
# >>> 희철
```

**확장자 필터링** — `endswith()`는 문자열이 특정 값으로 끝나는지 **bool**로 반환한다.

```python
리스트 = ["ab.xls", "a.hwp", "b.exe", "bbbbb.xls"]
for i in 리스트:
    if i.endswith(".xls"):
        print(i)
# >>> ab.xls
# >>> bbbbb.xls
```

| 메소드 | 역할 |
|------|------|
| `in` | 포함 여부 확인 |
| `endswith()` | 특정 문자열로 끝나는지 (bool) |
| `[:n]` (슬라이싱) | 범위 제한 |

---

## 3. 응용형 — 인덱스로 순회

자료구조 자리에 **인덱스 리스트**를 넣어 간접적으로 접근할 수도 있다.

```python
stocks = ["Naver", "Samsung", "SK Hynix"]
for 인덱스 in [0, 1, 2]:
    print(stocks[인덱스])
# >>> Naver
# >>> Samsung
# >>> SK Hynix
```

## 📝 정리

```
파이썬 반복문
├─ 기본형    for 변수 in 자료구조:  (콜론 + 들여쓰기)
├─ 슬라이싱  리스트[:2]로 범위 제한
├─ 조건      if 결합으로 원하는 값만 (endswith 등)
└─ 응용      인덱스 리스트로 간접 접근
```

| 개념 | 한 줄 정의 |
|------|------|
| **for ~ in** | 자료구조의 값을 하나씩 순회 |
| **콜론·들여쓰기** | 블록 구분(중괄호 대신) |
| **endswith** | 문자열 끝 검사 (bool) |

파이썬의 for는 "자료구조를 하나씩 꺼내 쓴다"는 점이 핵심이다. C/Java의 인덱스 기반 for보다 직관적이라, 자료구조와 함께 쓰면 아주 간결한 코드가 된다.
