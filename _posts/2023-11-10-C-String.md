## 📌 들어가며

C를 배우면서 문자열은 잠깐씩 봤지만, 알고 보면 꽤 복잡한 녀석이다. C에는 문자열 타입이 따로 없어서 **`char[]` 배열 + `'\0'`(널 문자)**로 다루고, 각종 처리는 `string.h`의 **`str___` 함수들**에 맡긴다. 이번 글에서는 대표적인 문자열 함수들을 정리한다.

> 이 글에서 다루는 함수 (모두 `#include <string.h>`)

| 함수 | 기능 |
|------|------|
| `strlen` | 문자열 길이(문자 개수) 반환 |
| `strcmp` | 두 문자열 비교 |
| `strcpy` | 문자열 복사 |
| `strcat` | 문자열 이어붙이기 |
| `strchr` | 문자열에서 '문자' 검색 |
| `strstr` | 문자열에서 '문자열' 검색 |
| `strupr` / `strlwr` | 대문자 / 소문자 변환 |

---

## 1. strlen — 문자열 길이

```c
strlen(문자열포인터);
strlen(문자배열);
```

![Desktop View](/assets/img/Programming-Language/C/String/1.png){: width="100%" }

`s1` 포인터에는 `"Hello"`의 주소를, `s2`에는 길이 10인 `char` 배열에 문자열을 할당했다. 문자열 끝에는 널 문자가 있으니, `strlen`을 쓰면 6이 나올까?

![Desktop View](/assets/img/Programming-Language/C/String/2.png){: width="100%" }

**5가 나온다.** `strlen`은 **널 문자(`'\0'`)를 제외한 순수 문자 개수**만 센다.

---

## 2. strcmp — 문자열 비교

```c
strcmp(문자열1, 문자열2);
```

`str` + `cmp(compare)`, 즉 두 문자열을 비교한다. 비교 기준은 **ASCII 코드**다. (예전에 `a`, `A`를 숫자로 나타냈던 그 고유값)

| 반환값 | 의미 |
|:---:|------|
| `-1` | ASCII 기준 **문자열2(s2)가 클 때** |
| `0` | 두 문자열이 **같을 때** |
| `1` | ASCII 기준 **문자열1(s1)이 클 때** |

![Desktop View](/assets/img/Programming-Language/C/String/3.png){: width="100%" }

`s2`에 `Helloo`처럼 문자가 하나 더 붙으면 `s2`가 더 크다.

![Desktop View](/assets/img/Programming-Language/C/String/4.png){: width="100%" }

`-1`이 출력되고, 작성한 `if`문대로 분기가 잘 동작한다.

---

## 3. strcpy — 문자열 복사

```c
strcpy(대상문자열, 원본문자열);   // 대상문자열의 포인터를 반환
```

![Desktop View](/assets/img/Programming-Language/C/String/5.png){: width="100%" }

`s1`에는 `Hello`를 넣고, `s2`는 초기화 없이 선언만 했다(→ 쓰레기값). `strcpy` 전과 후로 나눠 `s2`를 출력해 보자.

![Desktop View](/assets/img/Programming-Language/C/String/6.png){: width="100%" }

복사 전에는 이상한 값이던 `s2`가, 복사 후에는 `s1`의 `Hello`로 채워졌다.

> ⚠️ **주의**: `char *s1`처럼 **포인터로 선언된 문자열 리터럴**은 읽기 전용 데이터 영역(데이터 세그먼트)에 생성되어 **쓰기가 불가능**하다.
> ```c
> char *s1 = "Hello";
> char *s2 = "";
> strcpy(s2, s1);   // ❌ s2가 읽기 전용 영역 → 불가능
> ```

## 4. strcat — 문자열 이어붙이기

```c
strcat(최종문자열, 붙일문자열);   // 최종문자열의 포인터를 반환
```

`str` + `cat(concatenation)`, 문자열에 다른 문자열을 붙인다. 널 문자는 최종 문자열 끝에 한 번만 필요하므로, 이어붙일 때는 널 없이 문자만 이동한다.

![Desktop View](/assets/img/Programming-Language/C/String/7.png){: width="100%" }

`strcat(s2, s1);`은 **s2 뒤에 s1을 이어쓴다**는 뜻이다.

![Desktop View](/assets/img/Programming-Language/C/String/8.png){: width="100%" }

실행하면 `world`가 `Hello` 뒤에 붙는다.

---

## 5. strchr — 문자열에서 '문자' 검색

```c
strchr(대상문자열, 검색할문자);
// 찾으면 그 문자로 시작하는 문자열의 포인터 반환, 없으면 NULL
```

포인터를 반환하기 때문에, **찾은 문자 이후로도 쭉 출력**된다.

![Desktop View](/assets/img/Programming-Language/C/String/9.png){: width="100%" }

`ptr = strchr(s1, 'a');`는 `s1`에서 `a`가 처음 등장하는 위치의 주소를 담는다. `if(ptr)` 조건은 "못 찾으면 NULL(=0)"이므로 **찾았을 때만 참**이 된다. 이때 `ptr - s1`을 출력하면 **몇 번째에서 찾았는지**를 보여줄 수 있다.

![Desktop View](/assets/img/Programming-Language/C/String/10.png){: width="100%" }

이번에는 `while`문으로 만들었다. `ptr`이 NULL이 될 때까지 반복하되, 매번 `ptr + 1` 위치부터 다시 `'a'`를 검색해 **모든 'a'의 위치**를 찾는다.

![Desktop View](/assets/img/Programming-Language/C/String/11.png){: width="100%" }

`a`를 찾은 곳마다 위치와 그 이후 문자열을 출력했다. 복잡해 보여도 원리를 따라가면 재미있다.

---

## 6. strstr — 문자열에서 '문자열' 검색

```c
strstr(대상문자열, 검색할문자열);
// 찾으면 그 문자열로 시작하는 포인터 반환, 없으면 NULL
```

![Desktop View](/assets/img/Programming-Language/C/String/12.png){: width="100%" }

`strchr`이 '문자' 하나를 찾았다면, `strstr`은 **'문자열'을 통째로** 찾는다는 차이가 있다.

## 📝 정리

```
C 문자열 함수 (string.h)
├─ 길이    strlen  ('\0' 제외한 문자 개수)
├─ 비교    strcmp  (ASCII 기준, -1 / 0 / 1)
├─ 복사    strcpy  (대상 ← 원본)
├─ 붙이기  strcat  (대상 뒤에 이어붙임)
├─ 검색    strchr(문자) · strstr(문자열)  → 포인터 반환
└─ 변환    strupr / strlwr  (원본이 바뀜! 주의)
```

C의 문자열은 결국 **`char[]` 배열과 널 문자**의 조합이고, 각종 처리는 `string.h`의 함수들이 담당한다. 포인터를 반환한다는 점(`strchr`, `strstr`)과 원본을 바꾼다는 점(`strupr`, `strlwr`)만 잘 구분하면 헷갈리지 않는다.
