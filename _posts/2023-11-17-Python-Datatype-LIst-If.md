## 1. 숫자와 문자열

**숫자** — 천 단위 구분기호 없이 숫자만으로 구성해야 파이썬이 "숫자"로 인식한다.

```python
print(1000)     # int (정수)
print(100.0)    # float (실수)
```

**문자열** — 작은따옴표/큰따옴표로 묶인 문자들의 집합.

```python
"I'm a boy"    # 전체 따옴표에 영향 없는 다른 따옴표는 그냥 사용 가능
```

| 연산 | 문자열에서 |
|------|------|
| `+` | 이어붙이기 (**문자열끼리만**) |
| `*` | 숫자만큼 반복 복사 |

```python
print(77000 * 3, "원")   # 231000 원
```

**슬라이싱** — `[시작:끝]`, **시작은 '이상', 끝은 '미만'**이다.

```python
string = "hello minsu"
print(string[0:5])   # hello (인덱스 0~4)
```

> ⚠️ 문자열은 **한번 선언되면 수정 불가**(immutable)다. `string[0] = "H"`는 에러다.

## 2. 리스트

> **리스트**: 순서가 있는 자료구조. 대괄호 `[ ]`로 정의. 파이썬은 **다른 언어와 달리 타입이 통일되지 않아도** 되고(숫자·문자 혼합 가능), 각 원소를 **수정할 수 있다.**

```python
list1 = ['동희', '호로록']
list1[0] = "신동희"    # 문자열과 달리 수정 가능!
```

> 💡 리스트 원소가 수정 가능한 이유는, 각 원소가 **따로 메모리를 배정받아** 인덱스가 참조하기 때문이다.

**주요 메소드:**

| 메소드/키워드 | 역할 |
|------|------|
| `remove(값)` | 값으로 제거 |
| `pop(인덱스)` | 인덱스로 하나 제거 |
| `del` | 키워드로 제거 |
| `clear()` | 전체 제거 |
| `index()` | 값의 위치 |
| `in` | 존재 여부 확인 |
| `join()` | 구분자로 문자열 연결 |

## 3. Boolean과 조건문

Boolean은 참/거짓을 표현하며, 대부분의 언어와 동일하다.

```python
print(3 == 3)   # True
print(3 != 3)   # False
print(3 < 3)    # False
print(3 <= 3)   # True
```

**조건문** — 파이썬은 세미콜론 대신 **콜론(`:`)과 들여쓰기**로 블록을 구분한다.

```python
if 조건-1:
    print("state-1")
elif 조건-2:
    print("state-2")
else:
    print("state-N")
```

> 💡 인터프리터는 **위→아래**로 조건을 차례로 비교하고, 조건을 만족하면 그 블록을 실행한 뒤 **전체 if-elif-else를 종료**한다. 조건이 3개 이상이면 `elif`를 쓴다.

---

## 4. 논리 연산자

복잡한 조건을 비교할 때 **and / or / not**을 쓴다.

```python
print(True and True)           # True
print(True and True and False) # False
print(True or False)           # True
print(not True)                # False
```

| 연산자 | 참이 되는 조건 |
|:---:|------|
| `and` | 모두 True |
| `or` | 하나라도 True |
| `not` | 반전 |

## 📝 정리

```
데이터타입·리스트·조건
├─ 숫자/문자열  int/float, "" '', + 이어붙이기, * 반복
├─ 슬라이싱     [시작:끝] (시작 이상, 끝 미만)
├─ 리스트       [ ], 타입 혼합 O, 원소 수정 O
├─ 조건문       콜론(:) + 들여쓰기, if-elif-else
└─ 논리         and/or/not
```

| 개념 | 한 줄 정의 |
|------|------|
| **문자열 vs 리스트** | 수정 불가 vs 수정 가능 |
| **슬라이싱** | `[a:b]`, a 이상 b 미만 |
| **들여쓰기** | 파이썬의 블록 구분(세미콜론 대신) |

파이썬의 특징은 **들여쓰기로 블록을 구분**한다는 점, 그리고 리스트가 **타입 혼합·수정이 자유롭다**는 점이다. 이 두 가지 감각을 잡고 시작하자.
