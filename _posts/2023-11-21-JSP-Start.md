## 2. JSP 구성요소

JSP는 HTML 안에 자바 코드를 넣는 다섯 가지 요소로 이뤄진다.

| 요소 | 문법 | 역할 |
|------|:---:|------|
| **지시문(directive)** | `<%@ %>` | 페이지 속성 지정 |
| **선언문(declaration)** | `<%! %>` | **전역** 변수·메소드 |
| **스크립트릿(scriptlet)** | `<% %>` | 자바 코드 블록(**지역** 변수) |
| **표현식(expression)** | `<%= %>` | 결과값 출력(String) |
| **주석(comment)** | `<%-- --%>` | JSP 주석 |

**지시문 예:**

```jsp
<%@ page language="java" contentType="text/html; charset=UTF-8" %>  <!-- 페이지 속성 -->
<%@ include file="포함할 파일" %>  <!-- 다른 파일 포함(머리글/메뉴/꼬리글) -->
```

> 💡 **선언문 vs 스크립트릿**: 선언문(`<%! %>`)은 **전역** 변수·메소드를 선언(어디서든 사용 가능), 스크립트릿(`<% %>`)은 **지역** 변수 + 대부분의 자바 코드. 스크립트릿 안에서는 표현식(`<%= %>`)을 못 쓰고 **`out.print()`**를 쓴다.

---

## 3. 예제 — if문

```jsp
<%!
    int su1 = 10;   // 전역변수 (선언문)
    int su2 = 1;
%>
<%
    int result = 0, temp = 0;      // 지역변수 (스크립트릿)
    if (su1 >= su2) {
        result = su1 + su2;        // 크거나 같으면 덧셈
    } else {
        temp = su1; su1 = su2; su2 = temp;   // 교환 후
        result = su2 - su1;        // 큰 수 - 작은 수
    }
%>
연산 결과: <%= result %>   <!-- 표현식으로 출력 -->
```

## 📝 정리

```
JSP 시작
├─ 환경      Eclipse + Tomcat 9 → Dynamic Web Project
└─ 구성요소  지시문<%@%> · 선언문<%!%> · 스크립트릿<%%> · 표현식<%=%>
```

| 요소 | 한 줄 정의 |
|------|------|
| **지시문 `<%@%>`** | 페이지 속성 |
| **선언문 `<%!%>`** | 전역 변수·메소드 |
| **스크립트릿 `<%%>`** | 자바 코드 블록 |
| **표현식 `<%=%>`** | 값 출력 |

JSP는 "HTML 안에서 자바를 실행"하는 기술이다. 다섯 가지 구성요소(지시문·선언문·스크립트릿·표현식·주석)의 역할과 문법만 잡으면, 동적인 웹페이지를 만들 수 있다.
