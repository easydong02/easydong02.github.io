## 📌 들어가며

짧았던 C 언어 포스팅을 마치고, 이제 **Java**를 시작한다. 쉬운 길은 아니겠지만 그만큼 얻어갈 게 많을 거라 믿는다. 이번 글에서는 Java가 어떤 언어인지, 그리고 **JVM·JRE·JDK**의 구성과 기본 프로젝트 구조를 정리한다.

## 1. Java란?

**썬 마이크로시스템즈(Sun Microsystems)**에서 1995년에 개발한 **객체 지향 프로그래밍 언어**로, 창시자는 **제임스 고슬링(James Gosling)**이다.

| 항목 | 내용 |
|------|------|
| **개발** | 썬 마이크로시스템즈, 1995년 |
| **창시자** | 제임스 고슬링 |
| **저작권** | 2010년 오라클이 썬을 인수하며 소유 |
| **라이선스** | OpenJDK(GPL2) / Oracle JDK(상업용, 2019.1~ 유료화 강화) |
| **성향** | C#과 문법적으로 유사 |

**C++과의 차이** — C++은 기존 C에 객체지향을 얹다 보니 저수준·고수준 개념이 충돌하는 부분이 많았다. 반면 Java는 **처음부터 객체지향 언어로 설계**되었다. 다만 원시 타입(primitive)은 객체로 취급하지 않아, 모든 것을 객체로 보는 순수 객체지향 언어(Python, Ruby)와는 조금 다르다.

## 2. Java의 핵심 특징 — 플랫폼 독립성

Java의 가장 큰 특징은 **플랫폼에 독립적**이라는 점이다.

```
[C/C++]  소스코드 ──컴파일──► 기계어(플랫폼 종속)

[Java]   소스코드 ──컴파일──► 바이트코드(.class) ──JVM──► 기계어
                                   ↑ 어느 OS에서든 동일한 .class
```

C/C++ 컴파일러는 소스를 곧바로 **기계어로 컴파일·링크**하지만, 자바 컴파일러는 **바이트코드(`.class`)**를 생성한다. 이 바이트코드를 읽어 각 플랫폼의 기계어로 바꿔 실행하는 것이 **JVM**이다. 그래서 "한 번 작성하면 어디서나 실행(Write Once, Run Anywhere)"이 가능하다.

---

## 3. JVM · JRE · JDK

자바는 세 가지 구성 요소로 이루어진다. **JDK가 JRE를, JRE가 JVM을 포함**하는 구조다.

```
┌─────────────────────────────────────────┐
│ JDK (Java Development Kit)               │
│   개발에 필요한 도구(컴파일러 등) 포함     │
│  ┌────────────────────────────────────┐ │
│  │ JRE (Java Runtime Environment)     │ │
│  │   실행에 필요한 라이브러리·파일들    │ │
│  │  ┌──────────────────────────────┐  │ │
│  │  │ JVM (Java Virtual Machine)   │  │ │
│  │  │   Java 프로그램을 실행        │  │ │
│  │  └──────────────────────────────┘  │ │
│  └────────────────────────────────────┘ │
└─────────────────────────────────────────┘
```

| 구성 | 역할 |
|------|------|
| **JVM** | Java 프로그램을 실제로 **실행** |
| **JRE** | JVM 실행에 필요한 **라이브러리·파일** 보유 |
| **JDK** | JRE + **개발 도구**(컴파일러 등) 보유 |

> 💡 나는 **JDK 8**을 설치하고, IDE로 **Eclipse Photon**을 선택했다. 하나의 창에서 여러 프로젝트를 관리할 수 있어 편리하다.

## 4. 자바의 기본 구조

Java 코드는 다음과 같은 계층 구조로 조직된다.

```
프로젝트 ─► 패키지 ─► 클래스 ─► 메소드 ─► 소스코드
```

`day01` 프로젝트에 `day01` 패키지를 만들고, `PrintTest` 클래스 안의 `main` 메소드에 아래 코드를 넣어 이름을 출력해 보자.

```java
System.out.println("신동희");
```

## 📝 정리

```
Java 개요
├─ 정체    1995년 썬, 제임스 고슬링, 객체지향 언어
├─ 특징    플랫폼 독립 (소스→바이트코드(.class)→JVM→기계어)
├─ 구성    JDK ⊃ JRE ⊃ JVM
└─ 구조    프로젝트 ⊃ 패키지 ⊃ 클래스 ⊃ 메소드 ⊃ 소스코드
```

| 개념 | 한 줄 정의 |
|------|------|
| **바이트코드** | 플랫폼 독립적인 중간 코드(`.class`) |
| **JVM** | 바이트코드를 실행하는 가상 머신 |
| **JRE** | JVM + 실행 라이브러리 |
| **JDK** | JRE + 개발 도구 |
