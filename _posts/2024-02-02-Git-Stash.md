## 📌 들어가며

이번 글에서는 작업 중인 변경 사항을 **임시로 저장**하는 `git stash`를 정리한다. 커밋하기엔 애매한데 브랜치를 옮겨야 할 때, 변경분을 잠시 치워두고 나중에 되살리는 방법이다.

> **git stash란?** 현재 작업 중인 변경 사항을 **스택(stack)에 임시 저장**하는 명령. 여러 번 쌓을 수 있으며, 다른 브랜치로 이동하거나 급한 작업을 처리한 뒤 다시 꺼내 쓸 수 있다.

```mermaid
flowchart LR
    W["작업 중 변경"] -- "stash save" --> S[("Stash 스택")]
    S -- "apply/pop" --> W2["작업 디렉터리 복원"]
```

---

## 1. 핵심 명령어

| 명령 | 역할 |
|------|------|
| `git stash save "메시지"` | 변경 사항 저장(+메시지) |
| `git stash list` | stash 목록 확인 |
| `git stash apply stash@{n}` | 적용(stash 유지) |
| `git stash drop stash@{n}` | stash 제거 |
| **`git stash pop stash@{n}`** | 적용 + 제거(한 번에) |
| `git stash apply --index` | Staging Area까지 복원 |

```bash
git stash save "작업 임시 저장"
git stash list
git stash pop stash@{0}      # 적용 + 제거
```

> 💡 **`apply` vs `pop`** — `apply`는 stash를 **남겨둔 채** 적용하고, `pop`은 **적용 후 삭제**한다. 같은 변경분을 여러 브랜치에 적용할 거면 `apply`, 한 번 쓰고 말 거면 `pop`이 깔끔하다.

---

## 2. 실전 활용

### 다른 브랜치로 이동

```bash
git stash save "브랜치 전환용 저장"
git checkout feature-branch
git stash apply stash@{0}
```

### 급한 수정 처리

```bash
git stash save "Emergency fix"   # 하던 일 치워두고
# ... 급한 수정 ...
git stash apply stash@{0}        # 다시 원래 작업 복원
```

### 새 브랜치로 이어서 작업

```bash
git stash save "작업 이어가기"
git checkout -b new-feature-branch
git stash apply stash@{0}
```

> 💡 커밋하기엔 미완성이지만 브랜치는 바꿔야 하는 상황이 stash의 전형적 용도다. **작업물을 "잠깐 서랍에 넣어두는"** 느낌으로, 브랜치 간 이동을 자유롭게 해준다.

---

## 3. 주의사항

> ⚠️ **충돌 해결** — stash 적용 시 현재 코드와 충돌할 수 있다. 충돌을 해결하고 `git add` 후, 남은 stash는 `git stash drop`으로 정리한다.

> 💡 **유용한 옵션** — `--keep-index`(Staged 변경 유지), `--include-untracked`(추적 안 된 새 파일도 저장), `--patch`(변경 일부만 선택 저장). 필요에 따라 stash를 더 세밀하게 관리할 수 있다. 저장 시 **메시지를 명시**하면 나중에 어떤 작업인지 알아보기 쉽다.

---

## 📝 정리

```
git stash
├─ 저장   git stash save "메시지"
├─ 확인   git stash list
├─ 복원   apply(유지) / pop(제거)
└─ 옵션   --index / --include-untracked / --patch
```

| 개념 | 한 줄 정의 |
|------|------|
| **stash** | 변경 사항 임시 저장 스택 |
| **apply/pop** | 유지 복원 / 제거 복원 |
| **--include-untracked** | 새 파일까지 저장 |

git stash의 핵심은 **커밋하지 않고 작업물을 잠시 치워두는 것**이다. 브랜치를 자주 오가거나 급한 수정이 끼어들 때, 하던 일을 안전하게 보관했다가 되살릴 수 있어 매우 실용적이다.
